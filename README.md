# URLとは
**情報の住所**<br>
ある情報が「どこにあるのか」「どんなファイル名なのか」を示しています。<br>
例えばGoogleの検索画面は<br>
https://www.google.com/<br>
と表現されます。これは<br>
`https` = 通信プロトコル<br>
`www.google.com` = ホスト名<br>
`/` = パス<br>
と読み下すことができます。<br>
これは「`www.google.com`」というフォルダの「`/`」というページを表示してくださいね。その時の情報をやり取りする方法は「`https`」というルールに従ってくださいね。<br>
ということです。<br>
